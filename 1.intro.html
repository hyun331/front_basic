html 학습하기

h태그는 제목을 의미하고 위 아래로 약간의 여백이 자동 삽입됨

검색엔진에서 h태그는 중요한 검색 키워드로 사용됨

p 태그는 단락을 의미하고, 줄바꿈이 되면서 태그 위아래로 약간의 여백이 삽입

html 태그 내에서는 엔터를 치더라도 줄바꿈이 되지 않고,
br 태그를 써야함.

또한 웹브라우저에서는 띄어쓰기를 여러번 하더라도 한번만 띄어쓰기가 됨

텍스트의 형식화

굵게와 강조된 굵게와 이텔릭체와 강조된 이텔릭체 삭제된 텍스트를 표현할 때 del 태그 사용

html에서의 엔티티(특수기호) 사용

여러개의 공백을 넣고 싶을 때는     &+nbsp+;를 사용한다.(실제 사용에서는 +빼고)

단락을 표시하는 태그는 <p>태그이다.

태그를 화면에 표시하고 싶은 경우 & lt ; 태그 내용 & gt ;를 사용. 실제 사용에는 띄어쓰기 없이

div 태그는 특별한 기능이 없고, 다른 html요소들을 하나로 묶어 디자인 요소를 적용하는데 주로 사용. 줄바꿈됨.
span 태그는 특별한 기능이 없고, 마찬가지로 css 적용을 위해 사용. 줄바꿈 없음
